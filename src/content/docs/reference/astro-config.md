This site uses Astro with the Starlight docs theme.

## Configuration file

- Location: `astro.config.mjs`
- Export: `defineConfig({ integrations: [starlight({...})] })`

### Key options in this project

- **title**: Sets the site title displayed in the header.
- **social**: Adds social links (e.g., GitHub) to the site.
- **sidebar**:
  - A labeled group for **Guides** with a manual item.
  - An autogenerated group for **Reference** that mirrors files in `src/content/docs/reference/`.

### Example

```ts
// astro.config.mjs (excerpt)
import { defineConfig } from 'astro/config';
import starlight from '@astrojs/starlight';

export default defineConfig({
  integrations: [
    starlight({
      title: 'My Docs',
      social: [{ icon: 'github', label: 'GitHub', href: 'https://github.com/withastro/starlight' }],
      sidebar: [
        { label: 'Guides', items: [{ label: 'Example Guide', slug: 'guides/example' }] },
        { label: 'Reference', autogenerate: { directory: 'reference' } },
      ],
    }),
  ],
});
```

### Usage

- Add a new guide under `src/content/docs/guides/` and add it to the **Guides** sidebar group.
- Add a new reference page under `src/content/docs/reference/`. It will appear automatically under **Reference**.

### Tips

- See Starlight’s configuration docs for more options such as logo, i18n, search, and custom components.
